Step-1

There are 8 processes created, because whenever a fork() function called, it creates processes as child nodes of a growing binary tree. If fork() function is called two times, it will create processes. All these 4 processes will be the child node of binary tree.

If fork() function is called unconditionally l times, we will have 2l processes.

In this program fork() function is being called 3 times without any condition. So this program will generate total processes. There will be one parent process and seven child processes.
