Andrew File System (AFS) and Network File System (NFS)

Andrew file system has the servers of files and client computers whereas the network file system maintains the client and server aspect of AFS. The differences between the two are given as follows:

S. No.

AFS

NFS

1.

It contains the client computers and file servers.

NFS requires large space to access large files.

2.

File server run a process which is called vice.

NFS is less efficient as it requires more traffic in general to access the files.

3.

It uses its own customized protocol for the transfer of files and that protocol is referred as virtue.

In NFS a directory is mounted during boot process from the server.

4.

AFS has multiple set of features and more advanced.

NFS has less features and much simpler in designs.

5.

AFS differentiate between client and server and finds out the dedicated servers.

NFS permits a system to work as either a client or as a server.

6.

AFS maintains the state of updating of files and is stateful.

NFS is stateless and does not maintain the status of updating files.
