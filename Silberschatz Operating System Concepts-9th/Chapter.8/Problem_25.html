Step-1

Two memory accesses are required for a paged memory reference to reach the actual reference.

• Access the frame number on basis of page number

• Access the data on basis of frame number

The time taken for each memory reference is 50 nanoseconds.

The time taken to access the page table in RAM is 50 nanoseconds.

The time taken to access the data at the memory space is 50 nanoseconds.

Hence, the total memory reference time is nanoseconds.

Step-2

In case, translation look aside buffers (TLB) with 75% hit ratio are added, the effective memory reference time would be calculated as follows:

CASE 1: Page table entry is present in TLB:

The time taken for memory reference is equal to the sum of time taken to check TLB and time taken to access the data/Instruction.

Time taken to access a page-table entry in the TLBs=2 nanoseconds

Time taken to access the data/Instruction =50 nanoseconds

Hence, memory access time = nanoseconds.

Step-3

CASE 2: Page table entry is not present in TLB:

The time taken for memory reference is equal to the sum of time taken to check TLB, time taken to fetch page table entry from memory and time taken to access the data/Instruction.

Time taken to check TLB=2 nanoseconds

Time taken to fetch page table entry from memory=50 nanoseconds

Time taken to access the data/Instruction=50 nanoseconds

Hence, memory access time = nanoseconds.
