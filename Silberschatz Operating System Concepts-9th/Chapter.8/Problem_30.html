VAX architecture supports a variation of two – level paging. The VAX is a 32 bit machine with a page size of 512 bytes. The logical address space of process is divided into four equal section, each consists of bytes. Each section represents different part of logical address space of a process. The first 2 high – order bits of logical address designate the appropriate section, the next 21 bits represents the logic page number of that section, and final 9 bits represent an offset in desired page, By partitioning in desired page table, OS leave partition unused until a process needs them.
