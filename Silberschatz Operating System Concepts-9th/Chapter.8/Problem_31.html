Step-1

Comparison of segmented page scheme with hash page table is shown below:

S.no

Hash table

Segmentation

1

Hash table is used to store the large address spaces larger than 32-bits.

In segmentation, storage size is typically in thousands of bytes.

2

Logical address contains two fields which are page number and displacement or offset address.

Logical address is the collection of segments such as Subroutine, Stack, Symbol table, Sqrt, Main program.

3

Applying hash function to the page number gives the hash table entry.

Each segment has its own address space which is identified by two fields Segment-number and offset.

4

Virtual page number in the hash table is checked with the elements of the linked list. Page frame for the physical address is determined by matched element in the linked list.

Physical address can be determined from the segment table which contains segment base used to identify the starting address and segment limit used to specify the length of the segment.

5

Hash table entries are checked with linked lists which are in linear array format.

Segmentation is a user’s view of memory consists of unordered segments with various purposes.

6

Hash table limits the search to one or few page table entries.

Search is limited by base and limit pair in the segment table.

7

Bits are associated with page tables hence, protection is difficult.

Address is associated with logic unit (data, code and etc. are stored in segments). Hence, protection can be provided to each segment.

8

Variable allocation can’t be possible because, the size of the hash table can’t be varied.

Variable allocation is possible because segment length can vary.

9

Many pages map to the same entry, traversing the linked list incur significant overhead.

Segment table stores the address regarding to one page only. Hence, no overhead.

10

A hash page table is preferred when a program occupies only a small portion of its large virtual address space.

Segmented page scheme is preferred for large programs as the segment size can vary.
